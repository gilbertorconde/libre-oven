# Meireles MF 7606 X Oven Information

## Overview

The Meireles MF 7606 X is a built-in electric multifunction oven designed for versatile cooking. It offers a range of features to enhance your culinary experience.

## Key Features

- **Oven Type:** Electric multifunction
- **Functions:** 6 cooking functions, including:
  - Grill
  - Grill with top element
  - Bottom element
  - Top and bottom element
  - Top and bottom element with fan
  - Grill with bottom element and ventilation
- **Cooling System:** Tangential cooling fan
- **Controls:** Cylindrical knobs
- **Timer:** Minute minder
- **Interior Features:**
  - Lateral grids with anti-tilt system
  - Easy-to-clean enamel
  - Double removable glass door
- **Energy Efficiency:** Class A

## Technical Specifications

- **Oven Capacity:** 55 liters
- **Energy Consumption:**
  - Conventional heating: 0.79 kWh
  - Forced air heating: 0.75 kWh
- **Power Ratings:**
  - **Grill:** 1500 W
  - **Top element:** 1100 W
  - **Bottom element:** 1500 W
  - **Maximum power:** 2654 W
- **Electrical Requirements:** 220-240V / 50-60 Hz
- **Dimensions (W x H x D):** 598 x 580 x 500 mm
- **Packaging dimensions:** 665 x 670 x 580 mm
- **Weight:**
  - Net: 29 kg
  - Gross: 32 kg

## Function Mapping

According to official specifications, the oven has 6 functions:

1. **Grill** - 1500 W grill element
2. **Grill with top element** - Grill + Top element (1100 W)
3. **Bottom element** - Bottom element only (1500 W)
4. **Top and bottom element** - Both elements together (1100 W + 1500 W = 2600 W)
5. **Top and bottom element with fan** - Both elements + fan (convection mode)
6. **Grill with bottom element and ventilation** - Grill + Bottom + Fan
